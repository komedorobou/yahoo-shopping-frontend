## 🎯 主要機能

### 💰 Yahoo検索モード

- **完全自動検索**: CSVアップロード → 商品検索 → 利益計算まで全自動
- **リアルタイム進捗表示**: 検索状況をライブ表示
- **利益率フィルター**: 40%以上の商品だけ抽出
- **在庫状況チェック**: 在庫切れ商品を自動除外
- **並び替え機能**: 利益額/利益率/価格で自動ソート
- **チェックボックス選択**: 気になる商品を一括管理
- **外注先送信**: 選択商品をメール/LINEで送信

### 📊 CSV統合モード

#### 1. CSV結合機能
- **複数ファイル一括処理**: ドラッグ&ドロップで簡単アップロード
- **縦結合/横結合**: 2つの結合方法に対応
- **Excel対応**: .xlsx/.xlsファイルも読み込み可能
- **文字エンコーディング**: UTF-8/Shift-JIS自動検出
- **重複削除**: 重複行を自動削除

#### 2. B列クリーニング
- 「〇〇様専用」パターン削除
- 価格情報削除（定価〇円、¥〇〇など）
- 商品状態削除（美品、新品、未使用など）
- 宣伝文句削除（人気、激安、限定など）
- 過剰な記号削除（★☆♪◆など）
- カスタムパターン対応

#### 3. ブランド処理
- **300+ブランド自動認識**
- A列に自動抽出
- 主要ブランド:
  - Deuxieme Classe
  - IENA
  - Spick and Span
  - PLAGE
  - Noble
  - など300+ブランド

#### 4. 商品グループ化
- **類似商品を自動統合**
- 色名を無視（Black, Navy, Grayなど）
- サイズを無視（S, M, L, XLなど）
- 年代を無視（2023, 2024など）
- 類似度調整（30%〜100%）
- グループごとに価格分布を表示
- 中央値/最頻値/平均値を自動算出

### 👥 外注先管理機能

- **メール/LINE送信**: 利益商品を外注先に一括送信

## 📦 必要なもの

### 1. Yahoo Shopping API アプリケーションID（無料）
- [Yahoo! デベロッパーネットワーク](https://e.developer.yahoo.co.jp/register)で取得
- 1日50,000リクエストまで無料
- **🎥 特典: API取得マニュアル動画付き**（購入者限定）

### 2. メルカリCSVデータ

**2つの取得方法から選べる**

#### 方法A: オークファン活用（推奨）
- オークファンでメルカリ相場を検索
- CSVエクスポート機能でデータ取得
- 時間効率◎、精度◎

#### 方法B: 手動CSV作成（高精度）
メルカリで条件検索（例：「todayful コート 売り切れ 目立った傷や汚れなし 15000円以上」）
1. 商品画像から商品番号（例: 12020014）を取得
2. 価格情報も同時に取得
3. 200件程度のデータを手動収集
4. CSV統合モードでグループ化・分析

**メリット:**
- ✅ オークファン不要（コスト削減）
- ✅ 実売データで精度最高
- ✅ 一度作ったデータは資産として蓄積
- ✅ 忘れない、再利用可能

**必須列:**
- A列: ブランド名
- B列: 商品名
- D列: 価格（最頻値/中央値/平均値）

**推奨フォーマット:**
```csv
ブランド,グループ名,,価格最頻値
IENA,ラムウール ノーカラー ロングコート,,12000
Deuxieme Classe,ウールアンゴラリバーコート,,49000
```

## 📖 使い方

### 💰 Yahoo検索モード

1. **APIキー設定**: 初回起動時にYahoo Shopping APIキーを入力
2. **CSVアップロード**: メルカリ統合データをアップロード
3. **在庫フィルター**: 在庫不明を含めるか選択
4. **検索開始**: 「EXECUTE ANALYSIS」をクリック
5. **結果確認**: 利益商品がカード形式で表示
6. **並び替え**: 利益額/利益率/価格で並び替え
7. **商品選択**: チェックボックスで気になる商品を選択
8. **外注先送信**: 選択商品を外注先に送信

### 📊 CSV統合モード

#### ステップ1: CSV結合
1. 複数のCSV/Excelファイルをドラッグ&ドロップ
2. 文字エンコーディング選択（自動検出/UTF-8/Shift-JIS）
3. 結合方法選択（縦結合/横結合）
4. オプション設定（ヘッダー含める/重複削除）
5. 「CSVを結合」をクリック

#### ステップ2: B列クリーニング
1. 「🧹 B列クリーニング」をクリック
2. 削除パターンを選択
3. カスタムパターン追加（任意）
4. 「プレビュー」で確認
5. 「クリーニング実行」

#### ステップ3: ブランド処理
1. 「🏷️ ブランド処理」をクリック
2. 300+ブランドを自動認識
3. A列に自動抽出

#### ステップ4: 商品グループ化
1. 「📦 商品グループ化」をクリック
2. 類似度の閾値を調整（30%〜100%）
3. オプション選択:
   - 色名を無視
   - サイズを無視
   - 年代を無視
4. 「プレビュー」で確認
5. 「グループ化実行」

#### ステップ5: グループ集計
1. グループ化後に「📊 グループ集計」ボタンが表示
2. クリックで価格分布・中央値・最頻値を算出

#### ステップ6: ダウンロード
1. 「CSVをダウンロード」をクリック
2. 処理済みCSVをダウンロード
